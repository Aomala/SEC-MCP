Authentication

Include your API key in the X-API-Key header with every request.

curl -H "X-API-Key: YOUR_KEY" \ https://sec-mcp-production.up.railway.app/v1/financials/AAPL

Endpoints

All endpoints return JSON. Base URL: https://sec-mcp-production.up.railway.app

GET /v1/financials/{ticker} Get financial data from SEC filings
ParameterTypeDescription
tickerstring requiredStock ticker (e.g. AAPL, MSFT)
yearint optionalFiscal year (default: latest)
form_typestring optional10-K, 10-Q, 20-F, 6-K (default: 10-K)
Response
{ "ticker": "AAPL", "company": "Apple Inc.", "period": "FY", "year": 2024, "metrics": { "revenue": 391035000000, "net_income": 93736000000, "gross_margin": 0.462, "eps_diluted": 6.13, ... }, "filing": { "accession_number": "...", "filing_date": "..." }, "summary": "Apple reported $391B revenue..." }
GET /v1/compare?tickers=AAPL,MSFT,GOOG Compare multiple companies
ParameterTypeDescription
tickersstring requiredComma-separated tickers (max 10)
yearint optionalFiscal year
form_typestring optionalFiling form type
Response
{ "tickers": ["AAPL", "MSFT", "GOOG"], "results": [ { "ticker": "AAPL", "company": "Apple Inc.", "metrics": {...}, "summary": "..." }, ... ], "narrative": "AI comparison analysis..." }
GET /v1/search?q=apple Search companies by name or ticker
ParameterTypeDescription
qstring requiredSearch query
Response
{ "query": "apple", "results": [ { "ticker": "AAPL", "name": "Apple Inc.", "cik": "0000320193" }, ... ] }
GET /v1/cross-check/{ticker} Validate SEC data against Polygon.io

Cross-checks SEC XBRL extraction against Polygon's standardized financial data. Flags discrepancies >5%.

ParameterTypeDescription
tickerstring requiredStock ticker
Response
{ "ticker": "AAPL", "validation": { "revenue": { "sec": 391035000000, "polygon": 391035000000, "diff_pct": 0.0, "match": true }, "net_income": { "sec": 93736000000, "polygon": 93736000000, "diff_pct": 0.0, "match": true }, ... }, "polygon_details": { "name": "Apple Inc.", "market_cap": 3200000000000, ... } }
